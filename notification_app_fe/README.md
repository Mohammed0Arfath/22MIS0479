# notification_app_fe

Next.js frontend for viewing notifications.

Run locally:

```bash
cd notification_app_fe
npm install
npm run dev
```

App runs at `http://localhost:3000`.

Notes:
- The server-side pages read `stage1/token.json` (created during Stage 1) to authenticate requests to the notifications API.
